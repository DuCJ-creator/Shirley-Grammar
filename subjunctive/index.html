Introduction

什麼是虛擬語氣？

虛擬語氣 (Subjunctive Mood) 用來表達非事實、想像、願望或強烈建議。

核心概念：「時間退一步」。現在的假設用過去式，過去的假設用過去完成式。

1. If Conditions

與事實相反的假設
Present Unreal (與現在事實相反)
If + S + V-ed / were ..., S + would/could/might + V ...
  • If I were you, I would accept the offer.
    如果我是你，我就會接受這個提議。(事實：我不是你)
  • If I had enough money, I would buy that car.
    如果我有足夠的錢，我就會買那輛車。(事實：我現在沒錢)
Past Unreal (與過去事實相反)
If + S + had + Vpp ..., S + would/could + have + Vpp ...
  • If I had known the truth, I would have told you.
    如果我當時知道真相，我就會告訴你了。(事實：我當時不知道，也沒告訴你)
Future Unreal (與未來相反/萬一)
If + S + should / were to + V ..., S + would/could/will + V ...
  • If it should rain tomorrow, the picnic will be cancelled.
    萬一明天下雨，野餐就會取消。

2. Wish & Desire

願望與遺憾
Wish + 過去式 / 過去完成式

表達「希望但不可能發生」的事情。

  • I wish I were taller.
    我希望我能高一點。(現在的遺憾)
  • I wish I had studied harder last night.
    我真希望我昨晚有更努力唸書。(過去的遺憾)
  • If only I could fly!
    如果我會飛就好了！(語氣比 wish 更強烈)

3. The Mandative

命令、建議、要求

表示建議、命令、要求的動詞或形容詞後面的 that 子句，動詞必須用原形動詞 (Root Verb)，因為省略了 should。

S + V (suggest/order...) + that + S + (should) + V (root)
Key Words
Suggest Recommend Insist Order Demand Request Essential Vital Crucial
  • I suggest that he finish the work now.
    我建議他現在完成工作。(注意：不是 finishes)
  • The doctor insisted that she stay in bed.
    醫生堅持她臥床休息。(注意：不是 stayed)
  • It is essential that everyone be on time.
    每個人準時是至關重要的。(注意：用 be 而不是 are)

4. Special Cases

其他虛擬句型
As if / As though (彷彿)
  • He talks as if he knew everything.
    他講起話來好像他什麼都知道一樣。(其實不知道)
It is (high) time (該是...的時候了)
It is time + S + V-ed (過去式)
  • It is high time we went home.
    我們該回家了。(暗示動作有點晚了)
