The Logic of Order

Part III: Inverted Sentences · 結構的變奏

Strict Inversion

當否定詞 (Negative) 或 Only 位於句首時，句子必須倒裝，語氣強烈且正式。

  • Negative: Never have I... "Hardly did we arrive when it rained."
  • Only: Only then did I... "Only by practice can you succeed."

Conditional Shift

在虛擬條件句中，若省略 If，需將助動詞 (Had / Were / Should) 移至句首。

  • Had: Had I known... (= If I had known...)
  • Were: Were she here... (= If she were here...)

Rhetorical Effect

為了畫面感或強調程度，將地點副詞或 So/Such 置於句首引發倒裝。

  • Place: Here comes the bus. "Out rushed a big dog."
  • Degree: So fast did he run... "So moving was the speech that..."
