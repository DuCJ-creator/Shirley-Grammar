Part III: Inverted Sentences

Introduction

什麼是倒裝句？

倒裝句 (Inversion) 是指將句子中的動詞 (Verb) 或 助動詞 (Auxiliary) 移到 主詞 (Subject) 之前的句型。

目的：為了強調 (Emphasis) 語氣，或為了修辭的生動性。

正常語序：Subject + Verb ...
倒裝語序：Auxiliary / Verb + Subject ...

1. Negative Adverbs

否定副詞放句首
常用否定詞群

當以下否定副詞位於句首時，句子必須使用部分倒裝 (Partial Inversion)，即將助動詞移至主詞前。

Never Seldom Rarely Hardly Scarcely Barely Little Few Nowhere
Examples 例句
  • Never have I seen such a beautiful sight.
    我從未見過如此美景。 (原句: I have never seen...)
  • Seldom does she get angry.
    她很少生氣。 (原句: She seldom gets angry.)
  • Little did he know that the police were watching him.
    他一點也不知道警察正在監視他。
  • Nowhere can I find my wallet.
    我在任何地方都找不到我的錢包。
特殊句型：Hardly...when...
Hardly + had + S + Vpp... when + S + Vpt...
  • Hardly had we arrived home when it began to rain.
    我們一回到家，天就開始下雨了。

2. Only + Adverbial

Only 強調狀語放句首

當 Only 加上「時間副詞」、「介系詞片語」或「狀語子句」放句首時，主要子句需倒裝。

Only + (Time/Prep/Clause) + Aux + S + V...
Examples 例句
  • Only recently did I realize my mistake.
    直到最近我才意識到我的錯誤。
  • Only by working hard can you succeed.
    只有努力工作，你才能成功。
  • Only when you grow up will you understand.
    只有當你長大後，你才會明白。(注意：倒裝發生在主要子句)

3. Place & Direction

地方與方向副詞 (完全倒裝)

為了生動描述場景，將地點副詞或地方介系詞片語移至句首。此時多用完全倒裝 (Full Inversion)，即整個動詞移至主詞前。

Place/Direction + Verb + Subject
Here / There / Out / In / Down
  • Here comes the bus.
    公車來了。
  • Out rushed the students when the bell rang.
    鈴聲一響，學生們衝了出去。
注意：代名詞例外

如果主詞是代名詞 (Pronoun: he, she, it, they...)，則不倒裝。

  • Here it comes. (O)
  • Here comes it. (X)
Prepositional Phrases (介系詞片語)
  • Under the tree sat a boy.
    樹下坐著一個男孩。
  • On the hill stands a castle.
    山丘上矗立著一座城堡。

4. Conditional Sentences

省略 If 的假設語氣

在假設語氣中，可以省略連詞 If，並將助動詞 Were, Had, Should 移至句首。

三種情況
  • Type 1: Were (與現在事實相反)
    Were I you, I would accept the offer.
    = If I were you...
  • Type 2: Had (與過去事實相反)
    Had I known the truth, I would have told you.
    = If I had known the truth...
  • Type 3: Should (萬一)
    Should it rain tomorrow, the picnic will be cancelled.
    = If it should rain tomorrow...

5. So / Such ... that

程度副詞強調

為了強調形容詞或名詞的程度，將 So + adj/adv 或 Such + noun 移至句首，句子需倒裝。

  • So moving was his speech that everyone cried.
    他的演講是如此感人，以至於大家都哭了。
  • So fast did he run that no one could catch him.
    他跑得那麼快，沒人能抓到他。
  • Such was his anger that he lost control.
    他憤怒到失去了控制。
