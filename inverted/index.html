倒裝句介紹 | Inversion Introduction

什麼是倒裝句？ | What is Inversion?

英文句型的語序通常為S+V，但在一些情況下，會出現動詞放在主詞前的情況，就稱為「倒裝」。倒裝的最常見用法是疑問句 (如Are you busy?) ，這裏不做探討。

The usual word order in English sentences is S+V, but in some cases, the verb appears before the subject, which is called "inversion". The most common use of inversion is in questions (such as "Are you busy?"), which will not be discussed here.

正常語序 Normal Order: The technology has improved only in the past few years.

倒裝語序 Inverted Order: Only in the past few years has the technology improved.

倒裝句的基本結構 | Basic Structure of Inversion

倒裝句的基本結構 | Basic Structure of Inversion

特定詞語 + 助動詞/be動詞 + 主詞 + 主要動詞

Specific Word + Auxiliary/Be Verb + Subject + Main Verb

類型 Type 結構 Structure 例子 Example
Only 開頭 Only + 助動詞 + 主詞 + 主要動詞 Only then did I understand.
否定詞開頭 否定詞 + 助動詞 + 主詞 + 主要動詞 Never have I seen such beauty.
So 開頭 So + 形容詞 + 助動詞 + 主詞 + 主要動詞 So beautiful is she that everyone admires her.
地點副詞開頭 地點副詞 + 動詞 + 主詞 Here comes the bus.

Only 開頭 | Beginning with Only

Only 開頭的倒裝句 | Inversion Beginning with Only

當句子以"only"開頭，後面接副詞、副詞片語或副詞子句時，需要使用倒裝結構。

When a sentence begins with "only" followed by an adverb, adverbial phrase, or adverbial clause, inversion is required.

Only 倒裝結構 | Only Inversion Structure

Only + 副詞/副詞片語/副詞子句 + 助動詞 + 主詞 + 主要動詞

例子 Example: Only in the past few years has the technology improved enough for the price of computers to drop.

只有在過去幾年中，技術才進步到足以使電腦價格下降。

例子 Example: Only by practice can you make progress.

只有通過練習，你才能取得進步。

例子 Example: Only when he arrived did we start the meeting.

只有當他到達時，我們才開始會議。

例子 Example: Only then did I realize my mistake.

只有那時我才意識到我的錯誤。

Only 倒裝句的轉換 | Conversion of Only Inversion

Only 倒裝句可以從普通的陳述句轉換而來。

Only inversion sentences can be converted from ordinary declarative sentences.

正常語序 Normal Order: The technology has improved only in the past few years.

倒裝語序 Inverted Order: Only in the past few years has the technology improved.

正常語序 Normal Order: You can make progress only by practice.

倒裝語序 Inverted Order: Only by practice can you make progress.

正常語序 Normal Order: We started the meeting only when he arrived.

倒裝語序 Inverted Order: Only when he arrived did we start the meeting.

否定詞開頭 | Beginning with Negative Words

否定詞開頭的倒裝句 | Inversion Beginning with Negative Words

當句子以否定詞或帶有否定意義的詞語開頭時，需要使用倒裝結構。

When a sentence begins with negative words or words with negative meaning, inversion is required.

否定詞倒裝結構 | Negative Word Inversion Structure

否定詞 + 助動詞 + 主詞 + 主要動詞

常見否定詞 | Common Negative Words

絕不: never
絕不: by no means
絕不: under no circumstances
絕不: on no account
絕不: in no way
絕不: at no time
幾乎不: seldom
幾乎不: little
幾乎不: rarely
幾乎不: hardly
幾乎不: scarcely
直到…才…: not until
再也不: no longer
一…就…: no sooner …than…

例子 Example: Hardly did we see each other after we graduated.

畢業後我們幾乎沒有見過面。

例子 Example: Never should you trust him.

你絕不應該信任他。

例子 Example: By no means will I give up.

我絕不會放棄。

例子 Example: Scarcely had I left home when it started to rain.

我剛離開家就開始下雨了。

例子 Example: Not until 8pm did he come back home.

直到晚上8點他才回家。

例子 Example: Not only does he work hard himself, but he also helps others.

他不僅自己努力工作，還幫助他人。

對話 Dialogue:

--The girl can't swim.

--Nor/Neither can her sister.

--這個女孩不會游泳。

--她的姐姐也不會。

So 開頭 | Beginning with So

So 開頭的倒裝句 | Inversion Beginning with So

當句子以"So"開頭，後面接形容詞或副詞時，需要使用倒裝結構，表示"如此...以致於..."。

When a sentence begins with "So" followed by an adjective or adverb, inversion is required, meaning "so...that...".

So 倒裝結構 | So Inversion Structure

So + 形容詞/副詞 + 助動詞 + 主詞 + 主要動詞 + that 子句

例子 Example: So impressive was his progress in this test that the teacher awarded him a prize.

他在這次測試中的進步如此令人印象深刻，以致老師獎勵了他。

例子 Example: So quickly did he run that no one could catch him.

他跑得如此之快，以致沒有人能追上他。

例子 Example: So beautiful is the scenery that I can't take my eyes off it.

風景如此美麗，以致我無法移開視線。

對話 Dialogue:

-- I have been to the US.

--So have I.

--我去過美國。

--我也去過。

強調倒裝 | Emphasis Inversion

強調倒裝句 | Emphasis Inversion

為了強調某個部分，可以將該部分置於句首，並使用倒裝結構。

To emphasize a certain part, that part can be placed at the beginning of the sentence, and inversion is used.

例子 Example: Many a time has he helped me with my English study.

他多次幫助我學習英語。

例子 Example: There are birds singing in the tree.

樹上有鳥在唱歌。

例子 Example: Here comes the bus.

公車來了。

* 注意: Here he comes. 主詞為代名詞時，不用倒裝。

* Note: Here he comes. When the subject is a pronoun, inversion is not used.

例子 Example: Out rushed a big dog.

一隻大狗衝了出來。

例子 Example: Scattered on the floor were several books.

幾本書散落在地板上。

例子 Example: Tired as he was, we decided to give him a hand.

因為他太累了，我們決定幫他一把。

條件句倒裝 | Conditional Inversion

條件句的倒裝 | Conditional Inversion

在條件句中，如果省略"if"，則需要將助動詞提前，形成倒裝結構。

In conditional sentences, if "if" is omitted, the auxiliary verb needs to be moved forward, forming an inversion structure.

正常條件句 Normal Conditional: If you need any assistance, please call the reception.

倒裝條件句 Inverted Conditional: Should you need any assistance, please call the reception.

如果您需要任何幫助，請致電接待處。

正常條件句 Normal Conditional: If I had known he would be there, I would have not attended the meeting.

倒裝條件句 Inverted Conditional: Had I known he would be there, I would have not attended the meeting.

如果我知道他會在那裡，我就不會參加會議了。

(省略if, had 提前)

正常條件句 Normal Conditional: If she were here, she would help us.

倒裝條件句 Inverted Conditional: Were she here, she would help us.

如果她在這裡，她會幫助我們。
