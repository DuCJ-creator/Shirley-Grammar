1. Definition 定義

Adjectives (形容詞) Describe nouns (名詞).
e.g., a smart student.
Adverbs (副詞) Modify verbs, adjectives, or adverbs.
e.g., run fast, very good.

2. Adjectives Usage 形容詞用法

Position 位置

Attributive (前位): a successful businessman 🔊
Predicative (後位): The meeting is successful 🔊
💡 A-開頭形容詞: 只能放 be 動詞後面 (asleep, awake, alone, afraid)。

Compound Adjectives 複合形容詞

well-known three-year-old all-night good-looking

The + Adjective

The rich = Rich people (複數) 🔊

3. Adverbs 副詞形成

大多數副詞由 Adj + ly 構成，但要注意例外。

RuleExamples
Adj + lyquick → quickly
y → ilyhappy → happily
Irregular (同形)fast → fast, hard → hard
Irregular (變形)good → well
